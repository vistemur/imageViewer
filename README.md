# images preview app

## preview screen

- asynchronys image loading
- images are cashed
- download starts with device swithing to online
- tap failed images to reload
- tap loaded image to zoom (IOS 17 recommended)
- landscape and portrait modes supported

## image screen

- pinch to zoom
- swipe left and right to scroll through images
- tap to toggle fullscreen mode
- share links with button on top
